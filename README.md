# 삼장사 자소서 Assistant

이 프로젝트는 자기소개서 작성을 돕는 간단한 웹 챗봇입니다.

## 실행 방법
1. 브라우저에서 `index.html` 파일을 열어도 됩니다.
2. 또는 로컬 서버에서 실행할 수 있습니다.

## GitHub Pages 배포
1. GitHub 저장소에 이 파일들을 업로드합니다.
2. 저장소의 Settings → Pages로 이동합니다.
3. Source를 GitHub Actions로 선택합니다.
4. Actions 탭에서 `Deploy to GitHub Pages` 워크플로우를 실행합니다.
5. 워크플로우가 끝난 뒤, 저장소의 Actions 로그에서 배포 완료 상태를 확인합니다.
6. 배포가 완료되면 `https://<your-github-username>.github.io/<repo-name>/` 주소로 접속할 수 있습니다.

## 반영이 안 될 때 확인할 것
- 저장소에 최신 파일이 실제로 푸시되었는지
- GitHub Actions에서 워크플로우가 성공했는지
- Pages 설정이 GitHub Actions로 되어 있는지
- 브라우저 캐시를 새로고침했는지
